# 🏫 نظام إدارة الحضور المتطور

نظام شامل ومتطور لإدارة وتتبع حضور الأشخاص بواجهة مستخدم حديثة وسهلة الاستخدام.

## ✨ المميزات الجديدة

### 🎨 تصميم واجهة المستخدم المحدث
- **تصميم حديث ومهني**: واجهة مستخدم محدثة بالكامل مع ألوان متناسقة وخطوط واضحة
- **تجربة مستخدم محسنة**: تخطيط منظم وسهل الاستخدام مع انتقالات سلسة
- **تصميم متجاوب**: يعمل بشكل مثالي على جميع الأجهزة (الكمبيوتر، التابلت، الجوال)
- **نظام ألوان متقدم**: استخدام متغيرات CSS لسهولة التخصيص والصيانة

### 🔍 فلاتر المراحل
- **فلاتر ذكية**: أزرار فلترة للمراحل (الأولى، الثانية، الثالثة) في صفحات:
  - إضافة الأشخاص
  - تسجيل الحضور
- **فلترة فورية**: عرض الأشخاص حسب المرحلة المحددة دون إعادة تحميل الصفحة
- **عداد ديناميكي**: عرض عدد الأشخاص في كل مرحلة

### 👥 تحسينات صفحة إضافة الأشخاص
- **قائمة حية**: عرض جميع الأشخاص المسجلين أسفل نموذج الإضافة
- **تعديل مباشر**: إمكانية تعديل بيانات أي شخص بنقرة واحدة
- **نموذج تعديل متقدم**: تعديل الاسم، الجنس، والمرحلة مع حفظ فوري
- **إدارة النماذج**: عرض نموذج واحد فقط في كل مرة لتجنب التداخل
- **بطاقات أنيقة**: عرض بيانات الأشخاص في بطاقات منظمة مع شارات ملونة

### ✅ تطوير صفحة تسجيل الحضور
- **أسماء الأيام بالعربية**: عرض اليوم الحالي بالأسماء العربية (الاثنين، الثلاثاء، إلخ)
- **بطاقات الحضور**: تصميم بطاقات حديثة لكل شخص مع معلومات واضحة
- **حالة الحضور المرئية**: مؤشرات بصرية واضحة للحضور والغياب
- **ملخص فوري**: عرض إحصائيات الحضور والغياب في الوقت الفعلي
- **واجهة محمولة**: تصميم محسن للهواتف المحمولة

### 📊 تحسينات صفحة التقارير
- **منتقي التاريخ المتقدم**: استخدام حقول التاريخ بدلاً من القوائم المنسدلة
- **إحصائيات بصرية**: بطاقات إحصائية ملونة مع أرقام واضحة
- **تقارير مفصلة**: عرض نسبة الحضور وتفاصيل شاملة لكل شخص
- **طباعة محسنة**: زر طباعة عائم مع تنسيق مخصص للطباعة
- **جداول محسنة**: تصميم جداول حديث مع تأثيرات تفاعلية

### 🛠️ إصلاحات وتحسينات تقنية
- **إصلاح الأخطاء**: حل جميع مشاكل الفلاتر والنماذج المتداخلة
- **تحسين الاستعلامات**: تحديث استعلامات قاعدة البيانات لتشمل جميع البيانات المطلوبة
- **إدارة الحالة**: تحسين إدارة حالة النماذج والفلاتر
- **أداء محسن**: تحسين سرعة التحميل والاستجابة

## 🚀 كيفية الاستخدام

### 1. تشغيل التطبيق
```bash
# تهيئة قاعدة البيانات
python3 init_db.py

# (اختياري) إضافة بيانات تجريبية
python3 test_data.py

# تشغيل التطبيق
python3 app.py
```

### 2. الوصول للتطبيق
افتح المتصفح وانتقل إلى: `http://localhost:5000`

## 📱 الصفحات والمميزات

### 🏠 الصفحة الرئيسية
- مرحب بالمستخدم مع نظرة عامة على النظام
- بطاقات تنقل سريع لجميع الوظائف
- إحصائيات سريعة (قيد التطوير)

### ➕ إضافة الأشخاص
- نموذج إضافة شخص جديد مع الاسم، الجنس، والمرحلة
- فلاتر المراحل للبحث السريع
- قائمة حية بجميع الأشخاص المسجلين
- تعديل مباشر لبيانات أي شخص
- حذف الأشخاص مع تأكيد الأمان

### ✅ تسجيل الحضور
- عرض اليوم والتاريخ بالعربية
- فلاتر المراحل لسهولة التنقل
- بطاقات تفاعلية لكل شخص
- تسجيل الحضور بنقرة واحدة
- ملخص فوري للحضور والغياب

### 📊 التقارير
- **تقرير يومي**: حضور وغياب يوم محدد
- **تقرير شخصي**: سجل حضور شخص معين خلال فترة
- **إحصائيات مرئية**: بطاقات ملونة مع الأرقام
- **طباعة محسنة**: تنسيق مخصص للطباعة
- **حذف البيانات**: إمكانية حذف بيانات يوم كامل

## 🎨 التخصيص

النظام يستخدم متغيرات CSS لسهولة التخصيص:

```css
:root {
    --primary-color: #3b82f6;      /* اللون الأساسي */
    --accent-color: #10b981;       /* لون التمييز */
    --danger-color: #ef4444;       /* لون التحذير */
    --text-primary: #1f2937;       /* لون النص الأساسي */
    --border-radius: 8px;          /* انحناء الحواف */
}
```
